# variant-calling-analysis
Variant Calling Pipeline with Nextflow

Software needs:
- GATK4
- BWA
- Picard Tools
- Samtools
- SnpEff
- R (dependency for some GATK steps)
- Nextflow

Also: 
- Java
- bcftools
- bedtools
======================================

Pipeline is developed for exome sequencing data and works with curated gene list (relevant to Alzheimer's disease)
